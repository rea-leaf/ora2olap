# ora2olap
ora2olap是一款专注于 Oracle 到 OLAP 生态的表结构迁移工具。它能够智能解析 Oracle 数据字典，自动生成适用于 ClickHouse、Apache Doris 等主流 OLAP 引擎的建表语句（DDL），解决异构数据库之间的类型映射、分区策略、索引转换等适配难题，让数据仓库架构升级更加高效、可靠

### 项目定位

Oracle → OLAP 的 DDL 转换桥梁

### 核心能力

| 能力            | 说明                                                         |
| :-------------- | :----------------------------------------------------------- |
| **Schema 解析** | 深度解析 Oracle 数据字典，提取表结构、字段类型、约束、分区、索引等元信息 |
| **类型映射**    | 内置 Oracle 到 ClickHouse/Doris 的类型转换规则库，支持自定义扩展 |
| **分区适配**    | 智能转换 Oracle 分区策略为 OLAP 引擎最优分区方案             |
| **DDL 生成**    | 输出可直接执行的 ClickHouse/Doris 建表语句                   |
| **批量处理**    | 支持库级、Schema 级批量转换，提升迁移效率                    |

### 适用场景

- 企业数据仓库从 Oracle 向 ClickHouse/Doris 迁移
- 异构数据库双写架构的表结构同步
- 数据湖仓一体建设中的元数据对齐
